# Historical Docs

本目录用于存放已经完成阶段使命、但仍值得保留的历史文档。

这些文档通常具有以下特征：

- 记录某一阶段的讨论、取舍和中间结论
- 仍有参考价值，但不再代表当前代码现状
- 不适合作为“当前实现说明”继续维护

当前放入本目录的文档包括：

- `prompt-progress-memory-reference.md`
- `postprocess-issue-draft.md`
- `memory/mvp-plan.md`
- `memory/long-term-fix-plan.md`

使用建议：

- 看当前实现状态，优先读 `docs/Yakumo/modules/`、`docs/Yakumo/prompt-development-plan.md`、`docs/Yakumo/dev/memory/progress.md`
- 看某一阶段为什么这样设计，再回到本目录查历史记录
